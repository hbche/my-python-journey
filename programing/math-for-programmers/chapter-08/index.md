## 8.1 根据体积计算平均流速

现在声明以下两个函数：

```py
def volume(t):
    return (t-4)**3 / 64 + 3.3
```

### 8.1.1 实现 average_flow_rate 函数

已知体积随时间变化的函数，我们计算指定时间内的平均流速：

$$
平均流速=\frac{体积变化量}{经过时间}
$$

对应代码如下：

```py
def average_flow_rate(v, t1, t2):
    return (v(t2)-v(t1))/(t2-t1)
```
