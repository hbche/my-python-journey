# 第 1 章 简介与动机

机器学习的数据基础是理解机器学习算法的基石。

## 1.1 为直觉寻找词语

机器学习面临的挑战：对于概念和语义存在模糊的词语，机器学习系统会将其抽象成不同的数学表概念。

机器学习系统的主要组成部分：数据、模型和学习所需的数学概念和基础。
- 数据一般指经过转换之后适合计算机程序识别的向量数据。
- 模型通常用于描述生成数据的过程，与手头的数据集类似。
- 学习是指利用现有数据集通过效能函数的计算结果来优化模型的某些参数。

## 1.2 阅读本书的两种方法

推荐两种方式：
1. 自下而上：从基础概念到高级概念。
2. 自上而下：从实际需求深入到更基础的要求。

### 1.2.1 第一部分 关于数学

1. 学习使用向量表示数据，用矩阵表示这些数据的表格。
2. 学习向量相似性。
3. 学习关于矩阵和矩阵分解的一些基本概念。
4. 学习量化"噪声"。
5. 学习梯度，找到最大化性能指标的参数。

### 1.2.2 第二部分 关于机器学习

1. 学习如何设计机器学习试验评估方案
2. 研究线性回归
3. 主成分分析法降维
4. 密度估计
5. 支持向量机SVM
